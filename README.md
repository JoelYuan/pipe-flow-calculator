# pipe-flow-calculator
提供管径和介质性质表，直接得到应该选用的流量设计表格


# 管径流量预估表程序使用说明

## 程序功能
该程序用于根据管径、介质类型和压力信息，自动计算推荐流速、体积流量和质量流量，并生成Excel格式的结果报告。

## 输入文件格式要求

### 支持的文件格式
- Excel文件 (.xlsx)
- CSV文件 (.csv)

### 表格结构要求
输入表格必须包含以下三列（列顺序固定）：
1. **管径** - 管道直径值（支持数值或带mm单位的格式）
2. **介质** - 流体介质名称（必须使用程序支持的介质类型）
3. **备注** - 可选，可包含压力信息（支持MPa或bar单位）

### 介质列支持的类型
程序支持以下介质类型，填写时必须使用以下准确名称之一：

#### 水及水溶液
- 自来水
- 循环冷却水
- 盐水

#### 蒸汽系统
- 饱和蒸汽
- 过热蒸汽
- 冷凝水回水

#### 气体介质
- 压缩空气
- 天然气
- 氧气

#### 特殊流体
- 液氨
- 硫酸
- 泥浆

#### 暖通专用
- 乙二醇溶液
- 热水
- 高温烟气

### 填写示例
```
管径,介质,备注
100,自来水,普通管道
80,压缩空气,0.6MPa
50,饱和蒸汽,1.0MPa
```

## 备注列压力信息提取规则
- 支持从备注中提取压力值（MPa或bar单位）
- 格式示例："压力0.8MPa" 或 "10bar压缩空气"
- 如未提供压力信息，默认使用0.5MPa进行计算

## 使用方法
1. 运行程序后，程序会弹出文件选择对话框
2. 选择符合要求的.xlsx或.csv格式输入文件
3. 程序处理完成后，会弹出保存对话框
4. 选择保存位置和文件名，程序会自动确保扩展名为.xlsx
5. 查看生成的结果文件，包含计算的流量和设计建议

## 输出结果说明
输出Excel文件包含以下列：
- 管径(mm) - 输入的管道直径
- 介质 - 输入的介质类型
- 备注 - 输入的备注信息
- 压力(MPa) - 提取或默认的压力值
- 推荐流速(m/s) - 根据介质计算的推荐流速
- 体积流量(m³/h) - 计算的体积流量
- 质量流量(t/h) - 计算的质量流量
- 介质类别 - 介质所属类别
- 设计建议 - 相应的设计参考建议

## 注意事项
1. 介质名称必须完全匹配支持的类型列表
2. 管径值应使用数值或带mm单位的格式
3. 确保输入文件有正确的列顺序和内容
4. 结果文件将保存为Excel格式，可直接在Excel中打开查看
